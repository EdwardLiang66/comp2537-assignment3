# COMP 2537 Assignment 3

Pokemon Memory Match game built with HTML, CSS, and JavaScript.

## Local Preview

Double-click `start-local-server.bat`, then open:

```text
http://localhost:5500
```

## Features

- Random Pokemon cards loaded from PokeAPI
- Easy, medium, and hard difficulty levels
- Card flipping and matching logic
- Win and game-over events
- Click, timer, and pair status display
- Theme selection
- One-time reveal power-up
